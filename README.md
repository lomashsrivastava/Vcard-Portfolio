## About The Project

This is a personal portfolio website for **Gaurav Srivastava**, an MBA (HR) Graduate. It showcases his professional background, skills, projects, and educational qualifications. The portfolio is fully responsive and designed with a modern aesthetic.

### Key Features
- **Responsive Design:** Optimized for Desktop, Tablet, and Mobile.
- **Sidebar Integration:** Quick access to contact info and social links.
- **Dynamic Content:** Tabs for About, Resume, Portfolio, Blog, and Contact.
- **Testimonials Section:** Showcasing feedback from colleagues and mentors.
- **Portfolio Filter:** Filter projects by category (HR Analytics, Recruitment, Strategy).

## Tech Stack

- **HTML5**: For semantic structure.
- **CSS3**: For styling, including custom properties (variables) and responsive design (Flexbox/Grid).
- **JavaScript**: For interactive elements like the sidebar toggle and navigation tabs.
- **Ionicons**: For high-quality icons.
- **Google Fonts**: Typrography (Poppins).

## Project Structure

- `index.html`: Main entry point structure.
- `assets/css/style.css`: Main stylesheet.
- `assets/js`: JavaScript files for interactivity.
- `assets/images`: Images and icons used in the site.

## Setup and Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/lomashsrivastava/Vcard-Portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Vcard-Portfolio
   ```
3. Open `index.html` in your browser to view the site.
